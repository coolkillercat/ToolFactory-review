API de Dados v3

MERCADO BITCOIN

Antes de utilizar as API's do Mercado Bitcoin, consultar aqui o disposto no termo de uso das API's.

Essa pgina descreve a API de Dados, uma forma automatizada de obter os dados de negociaes do Mercado Bitcoin, em geral de uso de desenvolvedores de software que desejam integrar essas informaes a outros sistemas. Por esse motivo, os mtodos tm CORS configurado de maneira a permitir chamadas de outros domnios. A documentao abaixo descreve, com exemplos, como utilizar a interface de dados.

Acesso API de Dados

O acesso API de Dados pblico, no necessrio criar uma conta tampouco autenticar.

Estrutura das Requisies

URL

necessrio fazer uma chamada HTTP, mtodo GET, para a URL https://www.mercadobitcoin.net/api/<coin>/<method>/

 method: Mtodo da API de Dados requisitado
Tipo: String
Domnio de dados:
ticker : resumo de operaes executadas
orderbook : livro de negociaes, ordens abertas de compra e venda

Respostas

As Respostas so retornadas no formato JSON.


Exemplo de URL

https://www.mercadobitcoin.net/api/BTC/ticker/

Mtodos da API

ticker

Descrio

Retorna informaes com o resumo das ltimas 24 horas de negociaes.

Resultado
 high: Maior preo unitrio de negociao das ltimas 24 horas.
Tipo: Decimal
 low: Menor preo unitrio de negociao das ltimas 24 horas.
Tipo: Decimal
 vol: Quantidade negociada nas ltimas 24 horas.
Tipo: Decimal
 last: Preo unitrio da ltima negociao.
Tipo: Decimal
 buy: Maior preo de oferta de compra das ltimas 24 horas.
Tipo: Decimal
 sell: Menor preo de oferta de venda das ltimas 24 horas.
Tipo: Decimal
 date: Data e hora da informao em Era Unix
Tipo: Inteiro
Exemplo de chamada:
https://www.mercadobitcoin.net/api/BTC/ticker/
Exemplo de Retorno:
{
    'ticker': {
        'high': 14481.47000000,
        'low': 13706.00002000,
        'vol': 443.73564488,
        'last': 14447.01000000,
        'buy': 14447.00100000,
        'sell': 14447.01000000,
        'date': 1502977646
    }
}

orderbook

Descrio

Livro de ofertas composto por duas listas: (1) uma lista com as ofertas de compras ordenadas pelo maior valor; (2) uma lista com as ofertas de venda ordenadas pelo menor valor. O livro mostra at 1000 ofertas de compra e at 1000 ofertas de venda.

Uma oferta constituda por uma ou mais ordens, sendo assim, a quantidade da oferta o resultado da soma das quantidades das ordens de mesmo preo unitrio. Caso uma oferta represente mais de uma ordem, a prioridade de execuo se d com base na data de criao da ordem, da mais antiga para a mais nova.

Resultado
 bids: Lista de ofertas de compras, ordenadas do maior para o menor preo.
Tipo: Array
[0]: Preo unitrio da oferta de compra.
Tipo: Decimal
[1]: Quantidade da oferta de compra.
Tipo: Decimal
 asks: Lista de ofertas de venda, ordenadas do menor para o maior preo.
Tipo: Array
[0]: Preo unitrio da oferta de venda.
Tipo: Decimal
[1]: Quantidade da oferta de venda.
Tipo: Decimal
Exemplo de chamada:
https://www.mercadobitcoin.net/api/BTC/orderbook/
Exemplo de Retorno
{
    "asks": [
        [10410.00006000, 2.09190016],
        [10420.00000000, 0.00997000],
        [10488.99999000, 0.46634897]
    ],
    "bids": [
        [10405.38258000, 0.00181000],
        [10393.84180000, 0.08387000]
    ]
}

trades

Descrio

Histrico de negociaes realizadas.

Resultado
 []: Lista de negociaes realizadas.
date: Data e hora da negociao em Era Unix
Tipo: Decimal
price: Preo unitrio da negociao.
Tipo: Decimal
amount: Quantidade da negociao.
Tipo: Decimal
tid: Identificador da negociao.
Tipo: Inteiro
type: Indica a ponta executora da negociao
Tipo: String
Domnio de dados:
buy : indica ordem de compra executora
sell : indica ordem de venda executora
Exemplo de chamada:
https://www.mercadobitcoin.net/api/BTC/trades/
Parmetros adicionais

A chamada tradicional do histrico ir retornar as ltimas 1000 negociaes. Para obter dados anteriores, necessrio utilizar outros parmetros conforme descrito e exemplificado abaixo:

tid ou since: Retorna at 1000 negociaes a partir do identificador da negociao informado.
Chamadas:
https://www.mercadobitcoin.net/api/coin/trades/?tid=<tid>
https://www.mercadobitcoin.net/api/coin/trades/?since=<since>
Exemplo: 
https://www.mercadobitcoin.net/api/BTC/trades/?tid=5700
https://www.mercadobitcoin.net/api/BTC/trades/?since=5700
from: Retorna at 1000 negociaes a partir da data informada.
Formato: Era Unix
Chamada: https://www.mercadobitcoin.net/api/coin/trades/<from>/
Exemplo: 
https://www.mercadobitcoin.net/api/BTC/trades/1501871369/
from-to: Retorna at 1000 negociaes entre o intervalo de timestamp informado.
Formato: Era Unix
Chamada: https://www.mercadobitcoin.net/api/coin/trades/<from>/<to>/
Exemplo: 
https://www.mercadobitcoin.net/api/BTC/trades/1501871369/1501891200/

Retorna resumo dirio de negociaes realizadas.

Tipo: String
Formato: AAAA-MM-DD, exemplo: 2013-06-20
opening: Preo unitrio de abertura de negociao no dia.
Tipo: Decimal
closing: Preo unitrio de fechamento de negociao no dia.
Tipo: Decimal
lowest: Menor preo unitrio de negociao no dia.
Tipo: Decimal
highest: Maior preo unitrio de negociao no dia.
Tipo: Decimal
volume: Volume de Reais (BRL) negociados no dia.
Tipo: Decimal
quantity: Quantidade da moeda digital negociada no dia.
Tipo: Decimal
amount: Nmero de negociaes realizadas no dia.
Tipo: Inteiro
