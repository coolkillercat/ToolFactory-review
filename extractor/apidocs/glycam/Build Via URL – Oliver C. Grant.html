Build Via URL

October 12, 2022 oliver Uncategorized

GlycamWeb can build structures passed through the URL.

This might be useful when:

  • You want to build structures using an automated script.
    • Note that many text-based browsers will not work.
  • You want to provide access to GlycamWeb structures from another web site (although note that we have an API now).

How to build via URL

1. Express the carbohydrate you want to build using the condensed glycam notation.

  • Click here to learn condensed notation

2. Build the URL

The URL is composed of two parts.

  1. The base address, composed of the URL.  In other words: “glycam.org/url/condensed/”
  2. The sequence of your sugar.

For example, if you wanted to build this sugar:

The URI would be:

glycam.org/url/condensed/DManpa1-2DManpa1-3DManpa1-6[DManpa1-3]DManpb1-4DGlcpNAcb1-4DGlcpNAcb1-OH

3. Enter the URI into the address bar in your browser and go!

Note: Support for the internal notation of the Consortium for Functional Glycomics and “uncondensed” glycam has been discontinued.
