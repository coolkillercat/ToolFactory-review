Group Activity Analytics API

DETAILS: Tier: Premium, Ultimate Offering: GitLab.com, Self-managed, GitLab Dedicated

Get count of recently created issues for group

plaintext GET /analytics/group_activity/issues_count

Parameters:

| Attribute | Type | Required | Description | | --------- | ---- | -------- | ----------- | | group_path | string | yes | Group path |

Example request:

shell curl --header "PRIVATE-TOKEN: <your_access_token>" "https://gitlab.example.com/api/v4/analytics/group_activity/issues_count?group_path=gitlab-org"

Example response:

json { "issues_count": 10 }
