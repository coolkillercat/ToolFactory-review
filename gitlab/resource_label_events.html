Resource label events API

DETAILS: Tier: Free, Premium, Ultimate Offering: GitLab.com, Self-managed, GitLab Dedicated

Resource label events keep track about who, when, and which label was added to (or removed from) an issue, merge request, or epic.

Issues

List project issue label events

Gets a list of all label events for a single issue.

plaintext GET /projects/:id/issues/:issue_iid/resource_label_events
